# Assets Directory

Esta pasta deve conter os ícones do aplicativo:

- `icon.ico` - Ícone para Windows (256x256 mínimo)
- `icon.icns` - Ícone para macOS
- `icon.png` - Ícone para Linux (512x512 recomendado)

## Como criar os ícones

1. Crie uma imagem PNG de 512x512 pixels
2. Use ferramentas online como:
   - https://cloudconvert.com/ (para converter PNG para ICO)
   - https://iconverticons.com/ (para criar ICNS para Mac)

## Sugestão de design

O ícone deve representar acessibilidade e Libras:
- Símbolo de acessibilidade (♿)
- Mãos em gesto de Libras
- Cores: azul e verde (cores associadas à acessibilidade)
